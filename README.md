- **Navigation Bar** - Sticky navigation with smooth scrolling

## Getting Started

### Local Development

1. Clone this repository:
```bash
git clone https://github.com/mguyesikelelwe-hash/Simpex-solutions.git
cd Simpex-solutions
```

2. Open `index.html` in your web browser or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

3. Visit `http://localhost:8000` in your browser

### GitHub Pages Deployment

1. Go to your repository Settings
2. Scroll to **Pages** section
3. Set **Source** to `main` branch
4. Click Save
5. Your site will be available at: `https://mguyesikelelwe-hash.github.io/Simpex-solutions/`

## Customization

### Update Company Information
Edit the following files to customize for your business:

- **index.html** - Update company name, services, and content
- **styles.css** - Change colors by modifying CSS variables:
  - `--primary-color`: Main brand color
  - `--secondary-color`: Secondary accent
  - `--accent-color`: Call-to-action button color

### Example Color Changes
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    --accent-color: #your-color;
}
```
